# Crop_K-Means
通过数据挖掘相关知识处理数据集后，再用K-Means无监督聚类算法将作物按照所给的属性分类

### 处理数据集
  通过KDE（核密度估计法），将小于密度最大值三分之一的点筛选出来并去掉，从而得到更为紧密的数据集。
